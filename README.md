# Bright-Coffee-Sales-Analysis
This case study objective is to analyse sales data from 3 coffee shops for the new CEO.
from the analysis critical insights were extracted, i found that the best selling product is Coffee and Team. i also found that the highest sales and revenue are generated from the morning sales and i found that the lowest sales are generated at night. i also found that from April to June the sales spike high with June been the highest, this tells me that as the temparature changes the demand for hot beverages prevails.

in this case study i used Miro to create a mind mapping/Brainstorming, i used Canva to create a project plan, i used SQL and DataBricks to process the data. i used excel to create Pivot tables and PowerBI to create the Dashboard.
